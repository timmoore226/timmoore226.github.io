About Me

Welcome to my page! I'm a web / application developer with about two years of experience working on corporate websites, maintanence, and applications. Most of my professional work is attributed to websites and applications used internally and not available to the public; an unfortunate circumstance when trying to create a portfolio of work one has accomplished.

The main technologies I have used are the basic web languages, HTML, CSS, JavaScript, as well as JQuery and Bootstrap for extra pizzazz. Using C# and ASP.NET on the backend, I've used MVC patterns to design and create a dashboard for corporate security managers to view high level information and reports pulled by database queries I've written in SQL. I also have experience integrating Bing maps into an interface for managers to view locations of security incidents and facilities all over the world within UPS' organization.

In my free time I have enjoyed diving into Swift and web development. I don't have any fully functional apps or websites (besides this one) published just yet, but my aim over the next several months is to kick it into high gear to really start building up my personal portfolio and keeping up with the latest development trends. Until I get around to updating this page with my various projects, feel free to peruse my GitHub profile to see what I've been playing around with lately.
